Introduction

This is an Idyll post for assignment 10.

In this assignment, I build an Idyll webpage includes:

  • Idyll’s built in components
  • embedded vega-lite plot
  • dataset : Spending at Trump Properties

data set information:

  • Where you got it :I got the data set form ProPublica Data Store– Politics
  • A brief description:According to the propublica, this data set is “Detail of spending at Trump properties by political campaigns and federal officials between 2015 and the present”. The data set shows the campaigns and agencies spent of Donald J. Trump for President,Republican National Committee,Trump Victory,America First Action, Inc,Great America Committee,Bill Shuster for Congress. Also the data set includes the spent of texpayer dollars.
  • What you are trying to show in the plot: in this assignment, I want to show where they spend the money(“source” in the dataset) and how they spend the money(“purpose_scrubbed” in the dataset)

I will mention what vega-lite mark(s) I used and why in the following section along with the plots.
